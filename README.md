
CRUD Application (Spring Boot + React + MySQL)

This project is a basic CRUD (Create, Read, Update, Delete) application built using:

Backend: Spring Boot

Frontend: React

Database: MySQL

The application allows users to be added, updated, deleted, and viewed.


## Features
Add new users

View all users

Edit existing users

Delete users
## Tech Stack
*Backend:

Spring Boot

Spring Data JPA

Spring Web

MySQL Connector

*Frontend:

React

Axios

Bootstrap

*Database:

MySQL

## Setup Instructions
Java 17+

Node.js & npm

MySQL Server
